## Objetivo

O objetivo deste projeto é aplicar conceitos de modelagem matemática para analisar o impacto do acúmulo de poeira em placas solares e demonstrar como um sistema automatizado de limpeza pode contribuir para a recuperação da eficiência energética e para a redução de custos de manutenção.

## Modelagem Matemática

### Função Exponencial
E(t) = E₀ · e^(-kt)

### Função Quadrática
f(x) = -a(x - h)² + k

### Função Logarítmica
E(n) = C · log(n + 1)
